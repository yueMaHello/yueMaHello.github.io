Smart Mirror

This project is to build a family-use smart mirror using Raspberry Pi. The user could see himself in the mirror while viewing the message displaying around the edges of the mirror. This smart mirror system supports multiple users and each user can choose his or her personalized main page displaying weather, stock, time, calendar and news. The mirror integrates fingerprint recognition, human detection, and voice control features.

Proactive Network Maintenance

How could the technicians detect potential network issues before the customers are impacted? How could we fix the network issues before they get worse? This application is the solution! I developed this cable field fault-detection application based on the CableLabs' PNM technology. The application was tested in the real world, and it was proved to be able to detect about 35% network issues before the users were impacted.
